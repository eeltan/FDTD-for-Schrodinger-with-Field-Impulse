Matlab programs to generate results for each figures in "Schrodinger Equation with Gauge-Invariant Field-Impulse Replacing Potentials and Computation Using LOD-FDTD Method", by E. L. Tan and D. Y. Heh, Aug 2024

-mainfig1.m, for figure 1

-mainfig2.m, for figure 2
--consists of sub program(s) LOD3D.m

-mainfig3.m, for figure 3
--consists of sub program(s) LOD3D_Estate0.m, LOD3D_Estate1.m, LOD3D_Estate3.m

-mainfig4.m, for figure 4
--consists of sub program(s) LOD3D_multistate.m, LOD3D_multistateCFLN.m

-mainfig5.m, for figure 5
--consists of sub program(s) LOD3D_dterr.m, expm3D_dterr.m, explicit3D_dterr.m, expm3Dexpl_dterr.m, LOD3D_dxerr.m, explicit3D_dxerr.m

-mainfig678.m, for figures 6, 7, 8
--consists of sub program(s) LOD3D_allIE.m, explicit3D_allIE.m

-maintable1.m, for table 1
--consists of sub program(s) LOD3D_CPU.m, explicit3D_CPU.m

-IErectsol.mat, data file for field-impulses and magnetic field of a rectangular solenoid 

-IErectsolbig.mat, data file for field-impulses and magnetic field of a rectangular solenoid (larger domain size)
